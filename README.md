# zer0 graffit1

## 一言で言うと
- MarkdownのファイルをHTML+MantineUIで表示するWebアプリ

## 主な機能
- タグ管理
- コメント

## 構成
- Next.js
- Mantine UI
- Giscus

## 起動手順

```
yarn install
yarn run dev
```

## コントリビュート
 - バグ報告・機能提案はIssueへ、コード変更はPRでお願いします。
